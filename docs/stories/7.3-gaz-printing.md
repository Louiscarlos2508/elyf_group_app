# Story 7.3: Gaz Printing Integration

## Context
Integration of receipt printing for the Gaz module, including cylinder tracking.

## Requirements
- Include cylinder ID and label on the receipt.
- Trigger print after successful sale submission.
- Standardized gas receipt template.

## Implementation Details
Integrated into `GasSaleFormDialog` using `GasPrintReceiptButton` and `GasReceiptTemplate`.
