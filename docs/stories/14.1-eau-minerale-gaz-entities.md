# Story 14.1: Eau Minérale & Gaz Entity Hardening

## Context
Ensuring that all domain entities in the `Eau Minérale` and `Gaz` modules have consistent audit trails and multi-tenancy support.

## Requirements
- Add `enterpriseId`, `createdAt`, `updatedAt`, `deletedAt`, and `deletedBy` to all core entities.
- Standardize `fromMap` and `toMap` methods.
- Implement `isDeleted` getter in each entity.
- Ensure `copyWith` handles the new audit fields.

## Implementation Details
Check `lib/features/eau_minerale/domain/entities/` and `lib/features/gaz/domain/entities/`.
Primary entities updated: `Machine`, `ExpenseRecord`, `Product`, `Sale`, `Customer`, `CustomerCredit`, `CreditPayment`, `StockItem`, `GazExpense`, `FinancialReport`.
