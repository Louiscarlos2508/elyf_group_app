# Shared Layer

Widgets, providers et modèles transverses aux modules.

- `presentation/screens/` – écrans transverses (splash, onboarding, erreurs).
- `presentation/widgets/` – composants UI réutilisables (badges, charts).
- `providers/` – état global léger (thème, connectivité, boot status).
- `mixins/` & `extensions/` – helpers UI.
