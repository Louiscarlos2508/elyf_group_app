# Greenfield Full-Stack Development

Workflow for building new full-stack applications from scratch.

## Steps

1. **Project Briefing** (PM)
   - Define problem, users, and MVP scope.

2. **PRD Creation** (PM)
   - Create full requirements with personas, metrics, and risks.

3. **Architecture Design** (Architect)
   - Technical decisions and system design.

4. **PO Validation** (PO)
   - Validate all artifacts for consistency.

5. **Sharding Documents** (PO)
   - Shard documents for IDE development.

6. **Story Creation** (SM)
   - Create prioritized stories.

7. **Implementation** (Dev)
   - Implement stories.

8. **QA Review** (QA)
   - Validate implementation.

9. **Repeat**
    - Continue for all stories.
