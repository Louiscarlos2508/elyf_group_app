# Greenfield UI/Frontend Development

Workflow for building new frontend applications and user interfaces from scratch.

## Steps

1. **UI Briefing** (PM)
   - Define user goals, target platforms, and core journeys.

2. **PRD Creation** (PM)
   - Frontend requirements and state management needs.

3. **UX Specification** (UX Expert)
   - Create UI/UX designs and design system foundations.

4. **Architecture Design** (Architect)
   - Component architecture and routing strategy.

5. **PO Validation** (PO)
   - Validate design consistency and technical alignment.

6. **Sharding Documents** (PO)
   - Shard documents for IDE development.

7. **Story Creation** (SM)
   - Create UI stories.

8. **Implementation** (Dev)
   - Build components and screens.

9. **QA Review** (QA)
   - Visual and functional validation.

10. **Repeat**
    - Continue for all stories.
